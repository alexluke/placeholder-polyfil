Placeholder Polyfil

Add HTML5 placeholder support to older browsers

Usage

Use Modernizr to load this script for browsers that do not have native support

Modernizr.load({
    test: Modernizr.input.placeholder,
    nope: 'placeholder.min.js'
})

Requires >= jQuery 1.6
